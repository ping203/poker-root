The number of blinds levels during which rebuys are allowed.
NOTE: if the tournament has add-ons enabled, make sure that there is a break after the level where the rebuys stop.
For example, if you want rebuys during the first hour and then add-ons, if each level is 10 minutes,
you should set this to "6" and make sure that the 7th level is a break.
Example: 10 for a rebuy cost of €10, usually the same as the buy-in.
