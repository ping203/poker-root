ABOUT

This archive contains a standalone bundle of the Cubeia Network packaged in an Apache Tomcat web application server (http://tomcat.apache.org).

Applications included are:
- Backoffice: administration web application for Cubeia Wallet and Accounting services
- Wallet/Accounting: general accounting and wallet services
- User service: user database services

API documentation for the REST services are included in the doc directory. 

All services are configured to use a bundled H2 database.

See the installation.txt file for instructions on how to install, configure and run this bundle. 

More information about Cubeia Network:
