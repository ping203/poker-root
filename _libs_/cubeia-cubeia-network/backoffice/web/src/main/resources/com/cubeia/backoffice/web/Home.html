Cubeia Network

The Cubeia Network is a back office suite of high performance services and administrative tools for accounting, user management and integration and complex event processing.

All components in the Cubeia Network suite is open source and available under the AGPL licence.

Service Overview

The list below outlines the major services and components of the Cubeia Backoffice Suite.

  • User service - user account management
  • Accounting service - high performance accounting service
  • Backoffice administration interface - web based administrative tools for the backoffice services.

Getting started

  1. Check out the documentation
  2. Download binaries/source code
  3. Follow the tutorials and you are up and running in no time!
